🏟️ Community Sports Ground Booking System

A web-based platform that simplifies and digitizes the process of booking community sports grounds. This system eliminates manual booking conflicts by providing real-time availability, structured booking workflows, and transparent reservation management.

📌 Problem Statement

In many residential communities, sports ground bookings are handled manually, leading to:

Double bookings

Scheduling conflicts

Lack of transparency

Poor record management

This project provides a centralized digital solution to streamline ground reservations efficiently.

🎯 Project Aim

To develop a user-friendly website that enables residents to reserve sports facilities seamlessly while ensuring fair usage and proper scheduling.

🚀 Key Features
🔐 User Authentication

Secure login system for residents

Personalized dashboard access

📅 Ground Availability Calendar

Real-time availability view

Interactive calendar interface

Conflict-free booking system

🏏 Sport & Time Slot Selection

Choose from available sports (e.g., Cricket, Football, Badminton, etc.)

Select preferred date and time slot

📝 Booking Request Form

Simple and structured booking form

Validation to prevent duplicate or overlapping bookings

✅ Booking Confirmation Page

Confirmation message after successful booking

Booking reference details

📜 Rules & Guidelines Section

Community sports ground usage rules

Fair play and booking policies

📖 Booking History Page

View past and upcoming bookings

Easy tracking of reservations
